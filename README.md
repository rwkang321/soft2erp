# soft2erp

soft2erp ERP backend project.

## Working folder

- Windows: `D:\java\workspace\soft2erp`
- WSL: `/mnt/d/java/workspace/soft2erp`

Project files are created directly under this folder. Existing planning documents such as
`pln_soft2erp.md` remain in the same folder as project references.

## Stack

- Java 21
- Spring Boot
- Spring MVC REST API
- Thymeleaf for admin/test/development utility pages only
- Gradle
- MyBatis
- Supabase PostgreSQL

## First implementation scope

- Common API response
- Common exception handling
- Common code API
- Login/authorization skeleton
- Item master CRUD
